Regionalisation & RFA

Pooling Spatial Data to Estimate Extremes at Ungauged Sites

Commercial Context & Application

In catastrophe modeling, we frequently need to estimate 1-in-100 or 1-in-250 year events at locations where we have little to no historical gauge data. Regional Frequency Analysis (RFA) solves the "ungauged basin" problem by pooling data from a spatially homogeneous region. Instead of trusting a single, noisy 20-year local gauge record to extrapolate a 250-year event, we leverage hundreds of years of combined regional data to build a robust, shared growth curve.

1. The Index Flood Method

The cornerstone of modern regionalisation is the Index Flood Method. It assumes that within a homogeneous region, the distribution of extreme events at any site is identical everywhere, except for a site-specific scale factor.

The Index Flood Equation

The quantile of return period $T$ at site $i$, denoted $Q_T(i)$, is expressed as:

$$Q_T(i) = \mu_i \cdot q_T$$

Where:

  • $\mu_i$: The "Index Flood" (usually the mean or median of the annual maximum series at site $i$).
  • $q_T$: The Regional Growth Curve (a dimensionless factor for return period $T$, shared across the entire region).
Expand for Theory: L-Moments over Conventional Moments

Regionalisation heavily relies on L-moments (linear combinations of probability weighted moments) rather than conventional product-moments (variance, skewness, kurtosis).

  • Conventional moments square or cube outliers, making them highly unstable for extreme value analysis on short datasets.
  • L-moments are linear, making them significantly more robust to outliers and providing tighter bounds for parameter estimation.

For a sorted sample $X_{1:n} \leq X_{2:n} \dots \leq X_{n:n}$, the first two L-moments are:
$l_1 = E[X]$ (Mean)
$l_2 = \frac{1}{2} E[X_{2:2} - X_{1:2}]$ (L-scale or dispersion)

2. Delineating Homogeneous Regions

We cannot simply group stations by proximity. A homogeneous region must exhibit similar statistical behavior (L-CV, L-skewness) driven by similar physical topography and climatology. Hosking and Wallis (1997) established a rigorous heterogeneity measure, $H$, to test if a proposed region is statistically valid ($H < 1$ is acceptably homogeneous).

3. Implementation Methodology

The standard RFA workflow defined by Hosking & Wallis consists of four strict steps:

  1. Screen the data: Remove erroneous gauge data using discordancy measures ($D_i$).
  2. Identify regions: Cluster sites using physical characteristics and test statistical homogeneity using the $H$-measure.
  3. Choose a distribution: Use an L-moment ratio diagram to select the best-fitting regional distribution (e.g., GEV, GLO, GPA) via the $Z$-statistic.
  4. Estimate extremes: Calculate the regional growth curve $q_T$ and scale it by the local index flood $\mu_i$.

Code Implementations

Select a language below to see how to calculate L-moments and fit a regional distribution.

import numpy as np
import lmoments3 as lm
from lmoments3 import distr

# Simulated Annual Maximum Series (AMS) for 3 sites in a region
site_A = np.array([45, 52, 66, 40, 71, 55, 60, 80, 48, 51])
site_B = np.array([30, 35, 42, 28, 48, 38, 41, 50, 33, 36])
site_C = np.array([85, 92, 110, 75, 120, 95, 105, 130, 88, 91])

region_data = [site_A, site_B, site_C]

# 1. Calculate the Index Flood (Mean) for each site
index_floods = [np.mean(site) for site in region_data]

# 2. Normalize the data by the Index Flood (create the regional pool)
pooled_normalized_data = []
for site, index in zip(region_data, index_floods):
    pooled_normalized_data.extend(site / index)

# 3. Calculate Regional L-moments
regional_lmoms = lm.samlmu(pooled_normalized_data)
print(f"Regional L-Moments: l1={regional_lmoms[0]:.3f}, l2={regional_lmoms[1]:.3f}")

# 4. Fit a Generalized Extreme Value (GEV) distribution to get the Growth Curve
gev_params = distr.gev.lmom_fit(regional_lmoms)
print(f"Regional GEV Parameters: {gev_params}")

# 5. Estimate 1-in-100 year event for Site B (Ungauged/Short Record)
T = 100
q_100 = distr.gev.ppf(1 - 1/T, **gev_params) # Regional Growth Factor
Q_100_site_B = index_floods[1] * q_100

print(f"Estimated 100-year event for Site B: {Q_100_site_B:.2f}")

library(lmomRFA)

# Simulated data format for lmomRFA (SiteID, Year, Value)
site_A <- data.frame(Site="A", Year=2001:2010, Val=c(45, 52, 66, 40, 71, 55, 60, 80, 48, 51))
site_B <- data.frame(Site="B", Year=2001:2010, Val=c(30, 35, 42, 28, 48, 38, 41, 50, 33, 36))
site_C <- data.frame(Site="C", Year=2001:2010, Val=c(85, 92, 110, 75, 120, 95, 105, 130, 88, 91))
region_df <- rbind(site_A, site_B, site_C)

# 1. Convert to an object suitable for RFA
reg_data <- regsamlmu(region_df)

# 2. Check Regional Heterogeneity (H-measure)
# H < 1 indicates a safely homogeneous region
heterogeneity <- regtst(reg_data)
cat("Heterogeneity (H1):", heterogeneity$H[1], "\n")

# 3. Fit a Regional Growth Curve (GEV)
rfit <- regfit(reg_data, "gev")
cat("Regional GEV Parameters:\n")
print(rfit$para)

# 4. Estimate Quantiles for return periods
quantiles <- regquant(c(0.9, 0.99), rfit) # 10-year and 100-year growth factors
cat("\nRegional Growth Curve q(T):\n")
print(quantiles)

# Calculate Q_100 for Site B (Index flood = mean of Site B)
mu_B <- mean(site_B$Val)
Q100_B <- mu_B * quantiles[2]
cat(sprintf("\nEstimated 100-year event for Site B: %.2f\n", Q100_B))
